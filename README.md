## Environment

- Xcode: version ≥ Xcode13.0
- OSX: version ≥ 11.0

## Usage
- clone this repo
- open **Aquarius.xcodeproj** by Xcode(Version >= 11.0)
- `Command` + `R`, after resolve  Swift Packges

```shell
git clone https://github.com/CrazyFanFan/Aquarius.git
cd Aquarius
open Aquarius.xcodeproj
```
